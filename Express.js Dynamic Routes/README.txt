A simple Express.js project demonstrating dynamic routing. Users can access profiles using URLs with dynamic parameters like /users/101 or /users/admin.

Installation

Clone the repository.

Navigate to the project folder.

Run npm install to install dependencies.

Usage

Start the server: node app.js or nodemon app.js

Access user profiles in your browser, e.g., http://localhost:3200/users/101

Notes
Uses a simple in-memory object as a database.
